## Contact List

Contact List is a simple web application to save your contact in order not to miss anything.

### Features
* Add new Contact 
* Set Contact priority
* Edit Contact 
* Delete single Contact 
* Delete all Contact 

### Technologies used:
* HTML
* CSS
* JavaScript/React.js
* Font awesome
* Netlify
